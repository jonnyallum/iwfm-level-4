# Release Notes

## Version 0.1.0

Release date: 2026-06-24

## Summary

This release establishes the IWFM Level 4 Equivalent FM Academy as a production-ready baseline for practical FM learning, Longleat interview preparation and FM Control Hub training content.

## Included

- Course orientation.
- Modules 1 to 8.
- Longleat Boardroom Simulation.
- 100-question Longleat interview bank.
- 50-scenario FM operational scenario bank.
- Navigation, tags, glossary, quick start and roadmap.
- FM Control Hub and Longleat interview usage pages.
- AI/RAG metadata across key learning content.
- Quality audit and fix list.

## Fixes in This Release

- Added missing AI/RAG metadata to early modules.
- Corrected README repository map for the interview and scenario bank folders.

## Known Limitations

- Modules 9 to 12 are still incomplete.
- Workbook, mock exam, final assessment and product mapping layers remain open.
- Large practice-bank files should be split in a later mobile-reading pass.
