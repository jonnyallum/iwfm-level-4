## Purpose

Use this repo as the training and assessment layer for FM Control Hub. The Compliance Bible explains duties and evidence; this repo turns them into lessons, scenarios, model answers and interview-ready language.

## Product Mapping

| IWFM content | FM Control Hub feature |
|---|---|
| Modules | Training modules |
| Scenario bank | Scenario-based quizzes |
| Model answers | AI tutor feedback baseline |
| Marking guides | Assessment rubric |
| Interview question bank | Interview preparation pathway |
| Tags | Learner progress and RAG metadata |

## Implementation Steps

1. Convert each module into a learner pathway.
2. Use the [FM operational scenario bank](20-scenario-bank/fm-operational-scenario-bank.md) for scenario quizzes.
3. Use marking guides as rubric criteria.
4. Link each lesson to relevant Compliance Bible pages.
5. Use [Tags](TAGS.md) for learner progress, filters and AI/RAG indexing.
6. Use [Longleat Boardroom Simulation](13-longleat-simulation/longleat-boardroom-simulation.md) as an advanced capstone.

## Minimum Product Readiness Rules

- Every quiz answer should map to a model answer or marking guide.
- Every scenario should test action, evidence, ownership and escalation.
- Every module should link back to Compliance Bible source material.
- Learner progress should track confidence and demonstrated competence, not only completion.
